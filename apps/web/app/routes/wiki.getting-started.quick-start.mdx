# Quick Start Guide

Welcome to GiveawayBot! This guide will help you create your first giveaway in just a few minutes.

## Prerequisites

Before you begin, make sure you have:

- A Discord server where you have administrator permissions
- The GiveawayBot added to your server

> [!TIP]
> If you haven't added GiveawayBot to your server yet, click the "Add to Discord" button on our homepage.

## Your First Giveaway

<Steps>
- **Invite the bot**: Add GiveawayBot to your Discord server
- **Run the command**: Use `/start` to create a new giveaway
- **Configure settings**: Set the prize, duration, and entry requirements
- **Launch**: Review and launch your giveaway!
</Steps>

## Basic Commands

Here are the essential commands to get started:

<DocTabs>
  <TabItem label="Create" icon={<DocIcon name="plus" />}>
    ```
    /start
    ```
    Creates a new giveaway with an interactive setup wizard.
  </TabItem>
  <TabItem label="List" icon={<DocIcon name="list" />}>
    ```
    /list
    ```
    Shows all active giveaways in your server.
  </TabItem>
  <TabItem label="End" icon={<DocIcon name="player-stop" />}>
    ```
    /stop <giveaway-id>
    ```
    Manually ends a giveaway and picks winners.
  </TabItem>
</DocTabs>
