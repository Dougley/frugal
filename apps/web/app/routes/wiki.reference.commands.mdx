# Command Reference

Complete reference for all GiveawayBot slash commands.

## Giveaway Management

### `/start`

Creates a new giveaway with an interactive setup wizard.

**Usage:**
```
/start
```

**Permissions Required:** `Manage Server` or `Giveaway Manager` role

---

### `/stop`

Manually ends a running giveaway and selects winners.

**Usage:**
```
/stop giveaway_id:<id>
```

**Parameters:**
- `giveaway_id` - The ID of the giveaway to end

---

### `/reroll`

Rerolls the winners for a completed giveaway.

**Usage:**
```
/reroll giveaway_id:<id> [winners:<number>]
```

**Parameters:**
- `giveaway_id` - The ID of the giveaway to reroll
- `winners` (optional) - Number of new winners to select

---

### `/list`

Lists all giveaways in the current server.

**Usage:**
```
/list [status:<active|ended|all>]
```

**Parameters:**
- `status` (optional) - Filter by giveaway status (default: active)

---

## Configuration

### `/settings`

Configure bot settings for your server.

**Usage:**
```
/settings
```

**Permissions Required:** `Manage Server`

---

### `/ping`

Check the bot's latency and status.

**Usage:**
```
/ping
```
